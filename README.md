# Айтишечка Квиз Бот

Telegram-бот для викторины по IT-тематике.  
Пользователь может выбирать категорию и сложность, отвечать на вопросы, смотреть свой профиль и статистику, а также предлагать собственные вопросы для последующей модерации.

## Возможности
- регистрация пользователя при первом запуске;
- выбор категории вопросов;
- выбор сложности;
- получение случайных вопросов из базы;
- учёт уже пройденных вопросов;
- просмотр профиля и статистики;
- настройка отображения ника и решённых вопросов;
- отправка собственных вопросов на модерацию;
- отправка жалоб на вопросы.

## Как работает бот
После команды `/start` бот регистрирует пользователя в базе данных, если его ещё нет, и показывает главное меню.  
Далее пользователь выбирает категорию, затем сложность, после чего бот подбирает вопрос из базы данных.  
Вопросы хранятся в MySQL и фильтруются по категории, сложности, статусу модерации и истории прохождения пользователем.  
Также бот позволяет сохранять новые пользовательские вопросы в базу со статусом модерации и учитывать жалобы на существующие вопросы.

## Используемые технологии
- **Python**
- **pyTelegramBotAPI (telebot)** — Telegram Bot API
- **PyMySQL** — подключение к MySQL
- **MySQL** — база данных
- **logging** — логирование работы бота
- **hashlib** — генерация callback-идентификаторов для категорий

## Структура данных
Бот работает с таблицами:
- `users` — пользователи и их настройки;
- `categories` — категории вопросов;
- `questions` — вопросы и варианты ответов;
- `passedquestions` — история решённых вопросов;
- `complaints` — жалобы пользователей на вопросы.

## Основные сценарии
1. Пользователь запускает бота.
2. Бот показывает главное меню.
3. Пользователь выбирает категорию и сложность.
4. Бот выдаёт вопрос из базы.
5. Результаты и статистика сохраняются.
6. Пользователь может предложить свой вопрос или отправить жалобу.
